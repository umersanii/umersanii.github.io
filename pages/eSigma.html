eSigma

  • View On Github (v.1.0+)

    • About:

    eSigma is a versatile and powerful Discord bot meticulously crafted with the Discord.py library, and its primary mission is to revolutionize your voice channel experience. With eSigma, you're in control. Customize your Discord server's voice capabilities to match your unique preferences and enjoy a level of personalization like never before.

    Features:

    • ◉ Voice Channels: The bot can sense if someone joins the VC and will play an audio segment you choose for the user.
    • ◉ Random Audios: Using !p command you can trigger bot's main functionality that plays random audio saved in "randomvc" directory. You can use !f to stop

    How to Use:

    The bot has to be set up in order to be used.

    • • The Audios that you want to play with !p command will go in "data/randomvc" folder.
    • • The Audios that you want to play when a user joins the vc will go in "data/inoutvc" folder. But you will have to make a subfolder here with the user's discord id and user's name for your own ease. So say the user's name is Bruce and he has a discord ID 123, so his folder name will be "123Bruce". If you don't know how to get discord user ID, refer to this
